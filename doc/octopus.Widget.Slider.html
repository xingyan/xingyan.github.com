Name Type Description
data Array 轮播图的图片数据 如果指定了children属性 则此参数失效
children Array 需要进行轮播的节点
width Number 轮播的宽度 建议初始化赋值 可以省掉一次获取宽度的repaint
height Number 轮播的高度 建议初始化赋值
dataField Object 图片模式下 数据的解析值 默认为{ title: "title", url: "url", image_url: "image_url" }
isLon Boolean 是否纵向轮播 true为纵向 否则为横向 默认false
isNewTab Boolean 当点击轮播图的行为为默认行为跳转时生效 true为新窗口打开 false在原窗口打开 默认true
autoPlay Boolean 是否生成后进行动画 默认为true
autoPlayTime Number 轮播图动画时的停留时间 单位ms 默认为4000ms
animationTime Number 轮播图单次动画运行时间 单位ms 默认为400ms
animationType String 轮播图的动化类型 默认为"ease-out"
loop Boolean 是否是循环轮播 默认为true
hasTitle Boolean 是否有轮播图下方的title区域
hasGizmos Boolean 是否有轮播图下方的选择区域
disScroll Boolean 是否阻止滚动条
